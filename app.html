Install widget
Add NovaDesk to your store in 2 minutes
Step 1 — Copy your widget code

Copy this code and paste it before the </body> tag on your website.

<!-- NovaDesk Widget -->
<script>
  window.NovaDesk = {
    token: 'nd_live_a1b2c3d4e5f6',
    agentName: 'Nova',
    color: '#c8f04d'
  };
</script>
<script src="https://novadesk-azure.vercel.app/widget.js"></script>
Step 2 — Add to your store
Shopify
Online Store → Themes → Edit code → Paste before </body> in theme.liquid
WooCommerce
Appearance → Theme Editor → footer.php → Paste before </body>
Any HTML website
Open your HTML file and paste the code before the closing </body> tag
Step 3 — Verify installation

Enter your store URL to check if the widget is installed correctly.
